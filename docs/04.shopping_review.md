# 쇼핑몰 연동

## 프로젝트 개요

본 프로젝트는 외부 쇼핑몰 API 또는 백엔드 서버와 연동하여 상품 목록, 상세 페이지, 장바구니, 결제 기능을 구현하는 것을 목표로 합니다.  
사용자는 상품을 조회하고, 상세 정보를 확인하며, 장바구니에 담고 결제를 진행할 수 있습니다.

### 🎯 주요 기능

* 상품 목록 조회 (API 연동)
* 상품 상세 페이지 구현
* 장바구니 기능
* 결제 기능
* 사용자 인터랙션 기반 UI 업데이트
* 비동기 데이터 처리

### 🔗 서비스 URL

http://localhost:3000  
(배포 시 실제 URL로 변경)

### 👨‍👩‍👧‍👦 참여 팀원

* 김우혁 (Frontend / API 연동 / UI 구현)

---

## 고객사 요구사항

### 📧 이메일 기반 요구사항

* 상품 목록을 조회할 수 있어야 한다.
* 상품 클릭 시 상세 페이지로 이동해야 한다.
* 장바구니에 상품을 추가할 수 있어야 한다.
* 장바구니에 담긴 상품 수량을 확인할 수 있어야 한다.
* 결제 버튼 클릭 시 결제 페이지로 이동해야 한다.

---

### 🎨 피그마 시안 기반 요구사항

* 헤더에 장바구니 아이콘과 상품 개수 표시
* 상품 카드 UI (이미지, 이름, 가격 포함)
* 상세 페이지에서 상품 정보 및 설명 표시
* 장바구니 페이지에서 상품 수량 변경 가능
* 결제 완료 페이지 제공
* 사용자 행동에 따라 UI가 즉각적으로 반영될 것

---

## 요구사항에 따른 기능 동작 확인

### ✅ 1. 상품 목록 조회

* API 호출을 통해 상품 목록이 정상적으로 렌더링됨
* 페이지 로드 시 자동으로 데이터 요청

---

### ✅ 2. 상품 상세 페이지 이동

* 상품 클릭 시 상세 페이지로 정상 이동
* URL 파라미터를 통해 상품 ID 전달

---

### ✅ 3. 장바구니 기능

* 상품을 장바구니에 추가 가능
* 헤더의 장바구니 아이콘에 상품 개수 반영
* 동일 상품 추가 시 수량 증가

---

### ✅ 4. 수량 변경 기능

* 장바구니에서 상품 수량 증가 / 감소 가능
* 페이지 이동 후에도 상태 유지 (상태 관리 기반)

---

### ✅ 5. 결제 기능

* 결제 버튼 클릭 시 결제 페이지 이동
* 결제 완료 페이지 정상 출력

---

### ⚠️ 추가 확인된 사항

* 새로고침 시 장바구니 데이터 초기화 (로컬 저장 미구현)
* 네트워크 지연 시 로딩 처리 미흡

---
