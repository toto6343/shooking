# 슈킹(SHOOKING) 온라인 쇼핑몰 - 상품 목록 페이지

## 프로젝트 개요

슈킹 온라인 쇼핑몰은 20~30대 젊은 세대를 타겟으로 한 신발 전문 쇼핑몰입니다. 모바일 환경에 최적화된 이미지 중심의 쇼핑 경험을 제공하며, 상품 사진이 돋보이는 직관적인 UI와 실시간 장바구니 기능을 통해 사용자에게 편리한 쇼핑 환경을 제공합니다.

**주요 기능:**
- 2컬럼 그리드 레이아웃으로 모바일/데스크톱 모두에서 최적화된 상품 목록 표시
- 이미지 중심의 상품 카드 디자인으로 시각적 효과 극대화
- 장바구니 담기/제거 토글 기능과 실시간 헤더 카운트 업데이트
- 즉각적인 시각적 피드백으로 향상된 사용자 경험 제공

**참여 팀원:**
- 개발자: 김우혁
- 고객사 담당자: 슈킹 온라인 영업팀 최필승 대리

## 컴포넌트 구조

### **App.jsx**
애플리케이션의 최상위 컴포넌트로, 장바구니 상태를 관리하고 Header와 ProductList 컴포넌트를 렌더링합니다.

**주요 기능:**
- 장바구니 상태 관리 (`cartItems`)
- 장바구니 토글 함수 (`toggleCart`) 제공

```javascript
const [cartItems, setCartItems] = useState([]);

const toggleCart = (productId) => {
  setCartItems(prev => 
    prev.includes(productId) 
      ? prev.filter(id => id !== productId)
      : [...prev, productId]
  );
};
```

### **Header.jsx**
상단 헤더 영역을 담당하며, 브랜드 로고와 장바구니 아이콘 및 실시간 카운트를 표시합니다.

**Props:**
- `cartCount`: 장바구니에 담긴 상품 개수

### **ProductList.jsx**
상품 목록을 2컬럼 그리드 레이아웃으로 렌더링하는 컨테이너 컴포넌트입니다.

**Props:**
- `products`: 상품 데이터 배열
- `cartItems`: 장바구니에 담긴 상품 ID 배열
- `onToggleCart`: 장바구니 토글 함수

### **ProductCard.jsx**
개별 상품의 정보를 카드 형태로 표시하는 컴포넌트입니다.

**Props:**
- `product`: 상품 정보 객체 (id, name, price, image)
- `isInCart`: 장바구니 담김 여부
- `onToggleCart`: 장바구니 토글 함수

**주요 기능:**
- 상품 이미지 표시 (1:1 비율)
- 상품명 및 가격 정보 표시
- 장바구니 담기/제거 버튼 (상태에 따라 색상 변경)

## 주요 기능 상세

### 1. 장바구니 담기 기능
- 버튼 클릭 시 상태가 "담기" ↔ "담김"으로 토글됩니다.
- 담긴 상태에서는 버튼 색상이 변경되어 시각적 피드백을 제공합니다.
- 헤더의 장바구니 카운트가 실시간으로 업데이트됩니다.

### 2. 반응형 레이아웃
- 모바일 퍼스트 접근 방식으로 설계되었습니다.
- 모든 화면 크기에서 2컬럼 그리드를 유지하여 일관된 사용자 경험을 제공합니다.

### 3. 이미지 중심 디자인
- 상품 사진이 카드의 대부분을 차지하도록 설계되었습니다.
- 최소한의 텍스트 정보로 깔끔한 인터페이스를 구현했습니다.

## 유의사항 및 알려진 이슈

### 현재 제한사항
- 장바구니 데이터는 페이지 새로고침 시 초기화됩니다. (로컬 스토리지 미적용)
- 상품 상세 페이지는 현재 구현되지 않았습니다.
- 실제 결제 기능은 포함되어 있지 않습니다.

### 알려진 이슈
현재 특별히 보고된 버그는 없으나, 추가 브라우저 호환성 테스트가 필요합니다.

### 향후 개선 예정 사항
- 장바구니 데이터 로컬 스토리지 저장 기능 추가
- 상품 필터링 및 정렬 기능 구현
- 상품 상세 페이지 개발
- 검색 기능 추가
- 단위 테스트 및 E2E 테스트 작성
