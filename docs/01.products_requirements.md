# 01. 상품(Products) 요구사항 사양서

## 1. 개요
브랜드와 가격, 스타일이 결합된 고도화된 상품 탐색 및 몰입 환경 제공.

## 2. 기능 요구사항

### 2.1 상품 목록 (Product List)
- **Supabase 연동**: 실시간 DB 데이터를 기반으로 상품 목록 렌더링.
- **다중 필터링**: 브랜드 다중 선택 및 가격 범위 슬라이더(0~300,000원) 필터.
- **무한 스크롤**: '더 보기' 기능을 통한 점진적 데이터 로드.
- **정렬 기능**: 최신순, 가격순(저/고), 이름순 정렬 지원.
- **자동완성 검색**: 초성 및 부분 일치 검색 기반의 자동완성 추천.

### 2.2 상품 상세 (Product Detail)
- **360도 뷰어**: 수평 드래그를 통한 상품 회전 및 디테일 확인.
- **이미지 확대(Zoom)**: 클릭 시 전체 화면 고해상도 이미지 오버레이.
- **사이즈 시스템**: 230~290mm 사이즈 선택 기능. 품절 시 '알림신청' 전환.
- **스타일 스냅**: 상품별 유저 코디 갤러리 연동 및 '좋아요' 기능.
- **리뷰 시스템**: 별점 및 텍스트 후기 작성/조회 (DB 영구 저장).
- **연관 상품**: 동일 브랜드의 다른 상품 추천 알고리즘.

## 3. UI/UX 명세
- **Framer Motion**: 페이지 전환 및 360도 뷰어 애니메이션 처리.
- **다크 모드 지원**: 사용자 테마 환경에 따른 시각적 대응.
- **스켈레톤 UI**: 로딩 시 시각적 레이아웃 유지.
