# 🛍️ SHOOKING 결제 모듈 프로젝트

## 📌 프로젝트 개요

**SHOOKING 결제 모듈(SHOOKING Payment Module)** 은 빠르고 안전한 결제 경험을 제공하기 위한 **React 기반 결제 모듈 프로젝트**입니다.
장바구니를 거치지 않는 **즉시 결제 플로우**를 통해 결제 단계를 최소화하고, 사용자 경험(UX)과 구매 전환율 향상을 목표로 합니다.

### 🎯 프로젝트 목적

* 카드 등록부터 결제 완료까지의 결제 플로우 구현
* 카드 정보 보안을 고려한 사용자 인터페이스 제공
* 재사용 가능한 결제 컴포넌트 구조 설계

### 🔑 주요 기능 요약

* 카드 등록 및 카드 선택 기능
* 카드 정보 마스킹 처리
* 결제 확인 모달 제공
* 결제 성공 / 실패 처리 및 오류 안내

> 이 결제 모듈은 **카드를 등록하고 선택하여 결제할 수 있도록 돕는 모달 기반 결제 컴포넌트**입니다.

## 🧩 컴포넌트 구조

```text
src/
 ┣ components/
 ┃ ┣ CardInput/            # 카드 번호, 유효기간, CVC 입력 컴포넌트
 ┃ ┣ CardList/             # 등록된 카드 목록 및 카드 선택 UI
 ┃ ┣ PaymentModal/         # 결제 확인 및 결제 진행 모달
 ┃ ┣ PaymentButton/        # 즉시 결제 버튼
 ┣ pages/
 ┃ ┗ ProductPage.jsx       # 상품 목록 및 결제 진입 페이지
 ┣ context/                # 결제 및 카드 상태 관리
 ┣ App.js
 ┗ index.js
```

### 주요 컴포넌트 역할

* **CardInput**: 카드 정보 입력 및 유효성 검사, 입력 값 마스킹 처리
* **CardList**: 등록된 카드 목록 조회 및 카드 선택 기능 제공
* **PaymentModal**: 결제 정보 확인 및 결제 진행 상태 관리
* **PaymentButton**: 상품 페이지에서 즉시 결제를 트리거
