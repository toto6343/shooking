# 🛒 쇼핑몰 장바구니 페이지 개발 프로젝트

## 📌 기본 정보

* **프로젝트명**: 월드넷홀딩스 슈킹 온라인 쇼핑몰 장바구니 페이지 개발
* **사용하게 될 기술**: React (사용자 UI 및 상태 관리), Figma (디자인 시안 참고)
* **문서 작성자**: 김우혁

---

## 📝 고객 요구사항 정리

이미지 내 메일 본문을 바탕으로 정리한 주요 요구사항은 다음과 같습니다.

1. **정확한 금액 계산**: 장바구니에 담긴 상품의 금액을 오차 없이 계산하여 화면에 표시해야 함.
2. **배송비 정책 적용**:
* 총 상품 금액이 **10만원 미만**일 경우: 배송비 **3,000원** 부과.
* 총 상품 금액이 **10만원 이상**일 경우: **배송비 무료**.


3. **네비게이션**: '뒤로가기' 버튼 클릭 시 메인 상품 목록 페이지로 이동해야 함.
4. **결제 프로세스**: '결제하기' 버튼 클릭 시 결제 모듈이 **모달(Modal)** 형태로 호출되도록 연동해야 함.

---

## 📋 기능 목록

### 1. 장바구니 상품 목록 조회 및 관리

* 담긴 상품의 이미지, 상품명, 옵션 정보 표시
* 각 상품별 수량 변경 기능 (수량 변경 시 실시간 금액 재계산)
* 상품 삭제 기능 (개별 삭제 및 선택 삭제)

### 2. 금액 계산 엔진 (Price Calculator)

* **총 상품 금액**: 선택된 상품들의 가격 × 수량 합산
* **배송비 자동 계산**:
* `Total < 100,000` ? 3,000원 : 0원


* **총 결제 예정 금액**: `총 상품 금액 + 배송비` 표시

### 3. 페이지 이동 및 UI 인터랙션

* **뒤로가기 기능**: 메인 상품 리스트 페이지로 라우팅
* **결제 모달 연동**:
* '결제하기' 버튼 클릭 시 외부 결제 모듈 또는 커스텀 결제 UI를 모달 창으로 팝업
* 결제 모달 오픈 시 배경 스크롤 방지 처리



### 4. 반응형 디자인 (디자인 시안 준수)

* 피그마 시안(`node-id=0-1`)에 정의된 레이아웃 및 컴포넌트 스타일 적용
* 상품 리스트 및 결제 요약 영역의 그리드 레이아웃 구현
