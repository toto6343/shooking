# 장바구니 프로젝트

## 프로젝트 개요
본 프로젝트는 상품을 장바구니에 담고 관리할 수 있는 웹 애플리케이션입니다.  
사용자는 상품을 장바구니에 추가하고 수량을 조절하며, 장바구니에 담긴 상품 목록을 확인할 수 있습니다.

이 프로젝트는 React 기반 프론트엔드 애플리케이션으로 컴포넌트 기반 설계와 상태 관리를 통해 장바구니 기능을 구현하는 것을 목표로 합니다.

---

## 프로젝트 목표
- React 컴포넌트 구조 설계 및 재사용성 향상
- 상태 관리를 활용한 장바구니 기능 구현
- Storybook을 활용한 컴포넌트 문서화
- 테스트 코드를 통한 안정적인 UI 검증

### 주요 기능
- 상품 목록 조회
- 장바구니에 상품 추가
- 장바구니 상품 수량 변경
- 장바구니 상품 삭제
- 장바구니 목록 확인
