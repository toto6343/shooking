# 장바구니 페이지 문서화 (Cart Page Documentation)

## 1. 프로젝트 개요

장바구니 페이지는 사용자가 선택한 상품을 확인하고 결제를 진행할 수 있는 페이지입니다.  
사용자는 장바구니에 담긴 상품 목록을 확인하고 총 금액을 계산한 뒤 결제할 수 있습니다.

### 주요 기능
- 장바구니에 담긴 상품 목록 확인
- 상품 가격 합계 계산
- 배송비 자동 계산
- 결제하기 기능
- 뒤로가기 버튼을 통해 상품 목록 페이지로 이동

### 페이지 URL
예시  
http://localhost:3000/cart

### 참여한 팀원
- 김우혁
