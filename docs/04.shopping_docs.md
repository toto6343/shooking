# 쇼핑몰 연동

## 프로젝트 개요

본 프로젝트는 외부 쇼핑몰 API 또는 백엔드 서버와 연동하여 상품 목록, 상세 페이지, 장바구니 등의 기능을 구현하는 것을 목표로 합니다.
사용자는 상품을 조회하고, 상세 정보를 확인하며, 장바구니에 담는 기본적인 쇼핑 경험을 할 수 있습니다.

### 🎯 주요 기능

* 상품 목록 조회 (API 연동)
* 상품 상세 페이지 구현
* 장바구니 기능
* 사용자 인터랙션 기반 UI 업데이트
* 비동기 데이터 처리 (API 통신)

### 👨‍👩‍👧‍👦 참여 팀원

* 김우혁 (Frontend / API 연동 / UI 구현)
* (추가 팀원 작성)

## 설치 및 실행 방법

### 1️⃣ 프로젝트 클론

```bash

cd shooking
```

### 2️⃣ 패키지 설치

```bash

```

### 3️⃣ 프로젝트 실행

## 주요 함수와 기능

### 📦 상품 목록 조회

* API 요청을 통해 상품 데이터를 가져옴
* `useEffect`를 활용한 비동기 처리

```javascript
useEffect(() => {
  fetchProducts();
}, []);
```

---

### 🔍 상품 상세 페이지

* 상품 ID 기반으로 상세 데이터 조회
* URL 파라미터 활용

---

### 🛒 장바구니 기능

* 상품 추가 / 삭제 기능
* 상태 관리로 UI 실시간 반영

---

### 🔄 API 연동

* Axios 또는 Fetch를 사용한 서버 통신

```javascript
const fetchProducts = async () => {
  const response = await axios.get("/api/products");
  setProducts(response.data);
};
```
