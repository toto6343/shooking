# 쇼핑몰 연동

## 프로젝트 개요

본 프로젝트는 외부 쇼핑몰 API 또는 백엔드 서버와 연동하여 상품 목록, 상세 페이지, 장바구니 등의 기능을 구현하는 것을 목표로 합니다.
사용자는 상품을 조회하고, 상세 정보를 확인하며, 장바구니에 담는 기본적인 쇼핑 경험을 할 수 있습니다.

### 🎯 주요 기능

* 상품 목록 조회 (API 연동)
* 상품 상세 페이지 구현
* 장바구니 기능
* 사용자 인터랙션 기반 UI 업데이트
* 비동기 데이터 처리 (API 통신)

### 🔗 서비스 URL

(배포 시 실제 URL로 변경)

## 컴포넌트 구조

### 주요 컴포넌트 설명

* **Header**

  * 장바구니 아이콘 및 상품 개수 표시

* **ProductList**

  * 상품 목록을 렌더링
  * API로부터 데이터를 받아 화면에 출력

* **ProductItem**

  * 개별 상품 카드 UI
  * 클릭 시 상세 페이지로 이동

* **ProductDetail**

  * 상품 상세 정보 표시
  * 장바구니 추가 기능 제공

* **Cart**

  * 장바구니 목록 표시
  * 상품 수량 변경 및 삭제 기능

* **Checkout**

  * 결제 진행 및 완료 페이지

## 주요 함수와 기능

### 📦 상품 목록 조회

* API 요청을 통해 상품 데이터를 가져옴
* `useEffect`를 활용한 비동기 처리

```javascript
useEffect(() => {
  fetchProducts();
}, []);
```

---

### 🔍 상품 상세 페이지

* 상품 ID 기반으로 상세 데이터 조회
* URL 파라미터 활용

---

### 🛒 장바구니 기능

* 상품 추가 / 삭제 기능
* 상태 관리로 UI 실시간 반영

---

### 🔄 API 연동

* Axios 또는 Fetch를 사용한 서버 통신

```javascript
const fetchProducts = async () => {
  const response = await axios.get("/api/products");
  setProducts(response.data);
};
```
