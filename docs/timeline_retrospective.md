# 🛍️ Shoeking 쇼핑몰 연동 프로젝트

## 📌 프로젝트 개요

Shoeking 프로젝트는 쇼핑몰 상품 데이터를 연동하고, 사용자에게 상품 목록 및 상세 정보를 제공하는 웹/앱 서비스입니다.
외부 쇼핑몰 데이터와의 연동을 통해 실제 서비스 환경과 유사한 구조를 구현하는 것을 목표로 했습니다.

* **목표**

  * 상품 목록 및 상세 페이지 구현
  * 쇼핑몰 API 연동 경험 확보
  * 유지보수 가능한 구조 설계 및 리팩터링 경험

---

## 🧱 시스템 아키텍처

```
[Client]
  ↓
[Frontend (UI / 상태관리)]
  ↓
[Backend API]
  ↓
[외부 쇼핑몰 API]
```

* **Frontend**

  * 상품 목록 조회
  * 상품 상세 페이지 렌더링
* **Backend**

  * 외부 API 데이터 가공
  * 클라이언트 요청 처리
* **External API**

  * 상품 데이터 제공

---

---

## 🚀 주요 기능

### 1. 상품 목록 페이지

* 상품 리스트 조회
* 썸네일, 가격, 이름 표시
* 페이지네이션 또는 무한 스크롤

### 2. 상품 상세 페이지

* 상품 상세 정보 조회
* 이미지, 설명, 가격 표시

### 3. API 연동

* 외부 쇼핑몰 API 데이터 fetch
* 데이터 가공 후 클라이언트 전달
