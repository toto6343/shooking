# 슈킹(SHOOKING) 온라인 쇼핑몰 - 상품 목록 페이지

## 📋 프로젝트 개요

**프로젝트명**: 슈킹 온라인 쇼핑몰 구축  
**작성자**: 김우혁  
**작성일**: 2026-02-05  
**고객사**: 슈킹 온라인 영업팀 최필승 대리

### 타겟층
20~30대 젊은 세대를 대상으로 한 신발 전문 쇼핑몰

### 프로젝트 목표
모바일 환경에 최적화된 이미지 중심의 쇼핑 경험 제공

## 🎯 핵심 요구사항

1. **Mobile-First UI**: 스마트폰 쇼핑 비중이 높은 타겟층을 고려한 모바일 최적화
2. **시각적 강조**: 상품명보다 상품 사진이 돋보이는 레이아웃 구성
3. **실시간 인터랙션**: 장바구니 담기 시 헤더 카운트 실시간 업데이트
4. **피드백 체계**: 고객사 테스트를 위한 배포 URL 제공

## ✨ 주요 기능

| 기능 ID | 기능명 | 설명 |
|---------|--------|------|
| F01 | 2컬럼 그리드 레이아웃 | 모바일/웹 공통 2열 배치로 가독성 확보 |
| F02 | 이미지 중심 상품 카드 | 상품 사진이 가장 잘 보이도록 카드 영역 설계 |
| F03 | 장바구니 담기 토글 | 담기 버튼 클릭 시 즉각적인 시각적 피드백 |
| F04 | 헤더 실시간 카운팅 | 장바구니 담긴 상품 개수를 헤더에 실시간 표시 |
| F05 | 외부 배포(테스트) | 고객사 검토를 위한 테스트 URL 제공 |

## 🛠 기술 스택

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: CSS Modules
- **State Management**: React Hooks (useState)

## 🎨 디자인 가이드

- **레이아웃**: 2컬럼 그리드 (모바일/데스크톱 공통)
- **상품 카드**: 이미지 우선, 최소한의 텍스트 정보
- **색상**: 브랜드 컬러 기반 (Figma 시안 참조)
- **반응형**: 모바일 퍼스트 접근

## 📱 주요 화면 및 기능

### 1. 헤더
- 브랜드 로고 (SHOOKING)
- 장바구니 아이콘 + 실시간 카운트 표시

### 2. 상품 목록
- 2컬럼 그리드 레이아웃
- 각 상품 카드 포함:
  - 상품 이미지 (1:1 비율)
  - 상품명
  - 가격
  - 장바구니 담기 버튼

### 3. 장바구니 담기 기능
- 버튼 클릭 시 상태 토글 (담기 ↔ 담김)
- 헤더 카운트 실시간 업데이트
- 시각적 피드백 (색상 변경)
